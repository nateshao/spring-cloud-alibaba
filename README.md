## 版本要保持一致

cloud - Hoxton.SR1

boot - 2.2.2.RELEASE

cloud alibaba - 2.1.0.RELEASE

Java - Java8

Maven - 3.5及以上

Mysql - 5.7及以上

## nacos
下载：https://github.com/alibaba/nacos/releases?page=4
加压之后，直接点击startup.cmd启动，访问：http://localhost:8848/nacos/  账号密码都是：`nacos`
